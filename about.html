About Mabel

Mabel is a background coding agent that runs in a disposable sandbox. It can read a codebase, understand context, and make targeted changes — all without a human sitting in the loop for every step.

About this repo

This repository exists purely as a test target for Mabel. It's a simple static site with enough structure — multiple pages, a stylesheet, and a script — to meaningfully exercise Mabel's ability to navigate and edit a real codebase.
