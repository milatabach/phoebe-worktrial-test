Mabel Test Repo

A sandbox for testing Mabel — a background coding agent.

What is this?

This is a test repository for Mabel, a background coding agent that runs in a disposable sandbox. The site gives Mabel a real codebase to read, navigate, and edit — just enough surface area to verify that it can understand context and make accurate changes.

It has an About page, a shared stylesheet, and a small script. Nothing more, nothing less.
