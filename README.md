# mabel-test-repo

A test repository for **Mabel** — a background coding agent that runs in a disposable sandbox.

This site exists purely to give Mabel something real to read, edit, and reason about. It's a simple static website (HTML/CSS/JS) with a few pages, a stylesheet, and some scripts — just enough surface area to test whether Mabel can navigate a codebase, understand context, and make accurate changes.
